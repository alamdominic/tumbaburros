## 🎯 Cómo usar este material

### Para consultas rápidas:

1. Abre cualquiera de los archivos `.html` en tu navegador
2. Usa **Ctrl + F** para buscar el concepto específico que necesitas
3. Los cheatsheets están organizados por secciones con colores para facilitar la navegación visual

### Para estudio profundo:

1. Revisa el archivo markdown `cheatsheet_python.md` en tu editor favorito
2. Consulta los libros en PDF para entender conceptos en mayor detalle
3. Practica los ejemplos de código que encuentres

### Para imprimir o compartir:

- Los archivos HTML están optimizados para compartir en redes o imprimir
- `cheatsheet_py_linkedin.html` tiene formato compacto ideal para posts
- `cheatsheet_python.html` incluye media queries para impresión en A3
